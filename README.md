
# Rocket Learning Simulator

An AI-powered rocket simulator and learning platform for teens.

## Features
- Rocket building simulator
- Mission mode with AI-generated challenges
- Space map with rocket and satellite tracking
- AI-generated lessons and quizzes
- Rocket saving and custom presets
- User profiles and achievements
- Tutorials and guided mode
- Advanced physics options
- Optional multiplayer and sharing

## How to Use
1. Open the simulator in your browser.
2. Build a rocket using the parts library.
3. Launch your rocket and complete missions.
4. Check the space map to track your rocket.
5. Complete AI-generated lessons to learn rocket science concepts.
